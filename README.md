# BellaDona Peluquería Website

This repository contains the source code for the BellaDona Peluquería website, a hair salon located in Barcelona.

## About BellaDona

BellaDona offers a professional and personalized hair care experience. Services include cuts, styling, coloring, treatments, and more. The salon is located at Aribau 271 bajos, 08021 Barcelona.

## Website Features

*   **Homepage:** Introduction to BellaDona, featured services.
*   **Services Page:** Detailed list of services offered with prices.
*   **Bookings Page:** Information on how to book appointments (currently via phone).
*   **About Me Page:** Information about Evelyn, the founder and stylist.
*   **Contact Page:** Contact details, address, map, and opening hours.
*   **Multilingual:** Content available in Spanish (default), Catalan, and English, switched using JavaScript.
*   **Responsive Design:** Adapts to different screen sizes.

## SEO Enhancements Implemented

Several Search Engine Optimization (SEO) improvements have been made, including:
*   Creation of `robots.txt` and `sitemap.xml`.
*   Addition of meta descriptions to all pages.
*   Optimization of page titles and header tags.
*   Improved image alt texts.
*   Incorporation of relevant keywords into content.
*   Implementation of Schema.org structured data for LocalBusiness (HairSalon).
*   Addition of `hreflang` tags to signal language variations.

## Running Locally

To view the website, simply open the `.html` files in a web browser. No special build process is required.
